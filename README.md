# Load-Balanced Remote Call Procedure Service
## .Net сервис вызова удаленных процедур (REST-call) c балансировкой нагрузки

На текущем этапе разработки сервис выполняет свой функционал частично.
- не реализована балансировка нагрузки
- служба логирования имплементирова только частично
- библиотека HTTP-Get (IOContent.dll) возвращает не упакованный контент (gzip, deflate, br, zstd)
- версия для Docker не функциональна.
