# .Net Load-Balanced Remote Procedure Call Service
## .Net сервис вызова удаленных процедур c балансировкой нагрузки
- поумолчанию работает как многопоточный Web/REST-сервер
- библиотека HTTP-Get (IOContent.dll) поддерживает все популярные алгоритмы сжатия: gzip, deflate, br, zstd

На текущем этапе разработки сервис выполняет свой функционал частично.
- пока не реализована балансировка нагрузки
- служба логирования имплементирова только частично
